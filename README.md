# SubHub · 订阅聚合工具

🌐 一个简洁优雅的订阅聚合工具，支持合并多个订阅链接和节点，自动去重，生成 v2rayN / Clash 订阅链接。

## 功能特性

- 🔗 合并多个订阅链接（用 `|` 分隔）
- 📝 支持手动输入节点（vmess://、trojan://、ss://、vless://）
- 🧹 自动去重
- 🎯 关键词过滤（包含/排除）
- 🚀 生成短链接，支持 v2rayN / Shadowrocket / Clash
- 📱 完美适配电脑端和手机端
